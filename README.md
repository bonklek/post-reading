Standalone Chromium extension for read-aloud controls on X/Twitter posts.

```powershell
pnpm install
pnpm run build
```

Load `dist/post-reading-chromium` as an unpacked extension from `chrome://extensions`.

See [docs/POST_READING_STANDALONE.md](docs/POST_READING_STANDALONE.md) for the
feature map, startup flow, settings reference, TTS engines, OCR behavior, and
build layout.
